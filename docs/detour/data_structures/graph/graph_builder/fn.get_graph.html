pub fn get_graph(streams: Vec<Vec<[f64; 3]>>, config: Config) -> DetourGraph
Expand description

Returns a detour graph when given a vector of preprocessed streams.

First, streams are converted into paths using get_paths. Paths that does not contain at least one trajectory are removed. Then, each path is inserted into a detour graph and the nodes are merged. The graph is then made acyclic and finally the edges are merged.
